# Sistema de Diseño: Gourmet Speed

## Identidad de Marca

DotaBURGUERS es un concepto de **restaurante de hamburguesas gourmet temático de Dota 2**. El sistema de diseño refleja la energía del juego combinada con la calidez de una experiencia gastronómica.

### Filosofía de Color

- **Rojo (#b90e0a)**: Energía, urgencia, pasión — evoca la intensidad del juego y el apetito.
- **Amarillo (#f1b80c)**: Calidez, optimismo, hambre — el dorado de las recompensas y la comida reconfortante.
- **Crema (#fbf8f3)**: Limpieza, fondo neutro — permite que los colores vibrantes destaquen sin fatiga visual.

### Tipografía

**Hanken Grotesk** (Google Fonts) — tipografía geométrica moderna con pesos del 400 al 900. Elegida por su legibilidad en pantalla y su personalidad bold que combina con la estética del juego.

### Inspiración de Diseño

El sistema está inspirado en **Material Design 3 (Material You)**, adaptado para un contexto de fast-food. Se usa el enfoque de:

- **Tokens de diseño**: Colores, tipografía y espaciado centralizados (ver [DESIGN-TOKENS.md](./DESIGN-TOKENS.md)).
- **Superficies y elevación**: Paletas de surface con múltiples niveles de contenedor.
- **Color dinámico**: Esquema primary/secondary/tertiary que permite variaciones controladas.
